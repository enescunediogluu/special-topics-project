═══════════════════════════════════════════════════════════════════════════════

  Output: ESCO URI + Hierarchy + Confidence 0.95
  
TIER 2: DBPEDIA ENRICHMENT (Technical Metadata)
  Input:  "Python" (with ESCO match)
  Output: + DBpedia URI + Technical metadata + HYBRID_CONFLICT
  
TIER 3: FALLBACK (DBpedia Only)
  Input:  "Python" (no ESCO match)
  Output: DBpedia URI + Metadata + Confidence 0.75
  
TIER 4: UNRESOLVED
  Input:  Unknown skill
  Output: Surface form preserved + Confidence 0.0
